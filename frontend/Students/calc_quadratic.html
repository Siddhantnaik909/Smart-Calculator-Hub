The Quadratic Formula

[Image of Quadratic Formula]

The Discriminant (D) is b² - 4ac.

  • If D > 0 → Two real, distinct roots.
  • If D = 0 → One real, repeated root.
  • If D < 0 → Two complex (imaginary) roots.
